Shell Source Configuration Options¶

Shell¶

shell.command

The shell command to run.

  • Type: string
  • Importance: high
block.ms

The time to block after executing the shell command before executing it again.

  • Type: int
  • Importance: high
topic

The topic to write messages to.

  • Type: string
  • Importance: high
